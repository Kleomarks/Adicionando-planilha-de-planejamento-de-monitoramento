# 🛡️ Projeto Conecta Sênior: Monitoramento e Cibersegurança

Este projeto apresenta uma solução de monitoramento residencial para idosos, focada na segurança física e na proteção de dados da rede familiar.

## 📋 Sobre o Projeto
Desenvolvido como aplicação prática dos estudos em **Análise e Desenvolvimento de Sistemas (ADS)** e conceitos da **Cisco Networking Academy**.

## 📊 Planejamento Estratégico

| Categoria | Item | Objetivo |
| :--- | :--- | :--- |
| **Infraestrutura** | Roteador Mesh | Garantir sinal estável em toda a casa |
| **Energia** | Nobreak | Manter câmeras ligadas mesmo sem luz |
| **Segurança** | VLAN / Segmentação | Proteger a rede contra invasores |
| **Acesso** | MFA (2 Fatores) | Só a família acessa as imagens |
| **Privacidade** | Máscara de Zona | Respeitar a intimidade nos quartos |
